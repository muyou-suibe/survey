# Survey Data Cleaning, Visualization And Report

本项目围绕一份关于 MIB 实践成果替代传统学位论文认知与偏好的问卷展开，完成了从原始数据清洗、结构化处理、描述统计可视化，到结果汇总与结论提炼的完整分析流程。仓库不仅提供可复现的脚本，也可以直接作为课程作业、项目汇报或结果展示材料使用。

## 一、项目目标

本项目的核心目标有三项：

1. 对原始问卷数据进行清洗和有效性筛选，确保后续统计口径一致。
2. 对单选题、多选题、排序题、李克特量表题和权重分配题进行描述统计与图形化展示。
3. 基于统计结果提炼受访者在实践经历、政策认知、成果偏好、评价标准与支持需求方面的主要结论。

## 二、项目结构

- module1_data_cleaning.py：读取原始 Excel，完成筛选、拆分、校验和中间表导出。
- module2_visualization.py：基于清洗结果生成图表，并输出描述统计汇总表。
- output/：存放全部中间结果、统计图和汇总结果。
- task_list.md：记录各题统计任务与验收标准。

## 四、分析流程

整个分析流程分为两个阶段：

### 1. 数据清洗阶段

module1_data_cleaning.py 完成以下工作：

- 读取原始问卷数据。
- 过滤无效问卷。
- 输出 cleaned_data.csv 作为主分析表。
- 输出 multi_choice_binary.csv 作为多选题分析宽表。
- 输出 q6_ranked.csv 作为排序题分析表。
- 输出 cleaning_report.txt 作为清洗日志。

### 2. 可视化与汇总阶段

module2_visualization.py 完成以下工作：

- 为 Q1、Q2、Q4、Q5 生成单选题条形图加饼图。
- 为 Q3、Q7、Q8、Q11、Q12、Q13 生成多选题水平条形图。
- 为 Q6 生成排序题加权得分柱状图。
- 为 Q9 生成雷达图与箱线图组合图。
- 为 Q10 生成平均权重堆积条形图。
- 输出 descriptive_summary.csv 作为总汇总表。

## 七、统计口径说明

- 单选题：剔除缺失值后，统计频数和频率。
- 多选题：以有效问卷总数为分母，统计各选项的选择人数和选择率。
- Q6 排序题：按第 1 选乘 3、第 2 选乘 2、第 3 选乘 1 计算加权总分。
- Q9 李克特量表：统计各维度均值及 1 至 5 分分布。
- Q10 权重题：计算五类评价主体的平均权重，并验证总体接近 100%。

## 八、核心结果分析

### 1. 样本基本情况

- 受访者以研一学生为主，占 78.8%；研二占 18.8%；延期仅占 2.5%。
- 在实践经历方面，44.4% 的学生已有持续 3 个月以上的实习或实践经历，35.0% 有但不足 3 个月，20.6% 尚未开始实习。

这说明样本主体主要来自培养过程前中段，且多数学生已经具备一定程度的实践接触，因此对“实践成果替代论文”的态度具有较强现实基础。

### 2. 政策认知与校内标准认知偏弱

- 对学位法相关政策，“听说过但不清楚具体内容”占 35.0%，“完全不了解”占 31.9%，“有所了解”占 28.8%，“非常了解”仅占 4.4%。
- 对学校 MIB 实践成果具体要求，“不太清楚”占 40.6%，“基本清楚”占 35.0%，“完全不清楚”占 16.9%，“非常清楚”仅占 7.5%。

这组结果表明，学生对政策和制度的认知仍然以模糊了解为主，信息不对称是后续推进实践成果改革的重要阻力。

### 3. 当前实践成果产出总体不足

- Q3 中“尚未产出任何材料”选择率最高，达到 45.0%。
- 已产生的材料主要集中在商业计划书或营销方案 21.9%、数据分析报告 20.0%、市场调研报告 19.4%。
- 客户案例分析和项目实施总结相对较少，分别为 9.4% 和 13.1%。

这说明学生虽然存在实践经历，但尚未稳定形成高质量、规范化的成果产出，实践经历与可提交成果之间仍有明显转化缺口。

### 4. 学生偏好的成果形式更贴近真实商业问题

- Q6 排序题中，加权得分最高的是国际商务案例分析 213 分。
- 其次是企业咨询报告 164 分和市场调研报告 162 分。
- 商业方案设计 100 分、项目实施总结 104 分处于第二梯队。
- 政策研究报告、产品或服务创新方案、投资可行性研究、数字化解决方案得分明显较低。

这表明学生更偏好与真实企业情境、问题分析和决策建议直接相关的成果形式，而对抽象度更高、专业门槛更强或资源要求更高的类型兴趣相对有限。

### 5. 选择实践成果的主要驱动力来自相关性与成就感

- 选择实践成果类型的首要原因是“与我的实习或工作内容直接相关”，选择率 68.8%。
- 其次是“能真正解决实际问题，有成就感”，选择率 58.8%。
- “相对容易获得数据和企业支持”占 42.5%。
- “对未来就业有帮助”占 39.4%。

说明学生对实践成果的接受，并非单纯出于替代论文的便利性，而更看重成果与真实工作场景的连接价值。

### 6. 学生对不适合作为学位成果的类型判断较一致

- 65.6% 的学生认为企业内部保密性过强、无法公开的材料不适合作为替代成果。
- 57.5% 认为缺乏数据支撑的主观建议或简单文献综述类报告不适合。
- 48.8% 认为纯粹的实习日志或工作总结不适合。
- 30.0% 认为团队合作项目中个人贡献难以界定的成果存在问题。

这说明学生对实践成果的基本质量边界已有较清晰判断，即必须兼顾真实性、可验证性、数据支撑和个人贡献可识别性。

### 7. 高质量实践成果的评价标准较为集中

Q9 十个维度均值均高于 3.7，说明受访者普遍认为这些维度重要。其中得分较高的维度包括：

- 问题导向性：4.28
- 真实性：4.19
- 可操作性：4.14
- 规范性：4.13
- 数据支撑度：4.11
- 实践价值：4.04

相对较低但仍然重要的维度为：

- 理论应用性：3.82
- 团队协作能力：3.83
- 创新性：3.74

这意味着学生更重视实践成果是否立足真实商业问题、是否有真实数据和实际应用价值，而对“创新性”与“理论包装”的优先级略低。

### 8. 评价主体权重更强调导师与外部实践方共同参与

Q10 平均权重结果为：

- 校内学术导师：26.0%
- 校外企业导师：21.0%
- 企业客户或受益方：20.0%
- 学院专家评审组：17.3%
- 学生自评：15.7%

可以看出，学生更倾向于建立“校内导师主导、企业导师与实践对象共同参与、学生自评占较小比例”的复合评价结构。

### 9. 学生的主要顾虑集中在标准不清与认可机制不明确

- 62.5% 的学生担心“不知道什么类型的实践成果能被学院认可”。
- 56.9% 担心“做了企业项目最后论文评审不通过”。
- 41.9% 不清楚实践成果的具体质量要求。
- 30.6% 反映企业不愿意配合或提供真实数据。
- 30.0% 认为缺乏模板和范例支持。

这说明当前的主要障碍并不是学生缺少意愿，而是规则、质量标准、评价机制和案例资源尚不够透明。

### 10. 对学院支持措施的需求非常明确

- 在分类标准制定方面，学生最关注与 MIB 专业方向的匹配度 61.9%、与实习岗位和职业规划的相关性 56.9%、成果类型的多样性和灵活性 48.1%。
- 在支持工具方面，学生最希望获得不同类型实践成果的撰写模板 66.2%、优秀案例库 58.1%、企业项目对接平台 54.4%、评价指标细则和评分表 48.8%。

这表明制度建设不能只停留在“允许替代”的原则层面，还需要配套模板、案例、平台和细则，才能真正提升实践成果方案的可操作性。

## 九、结论与建议

基于本次问卷结果，可以形成以下结论：

1. 学生对实践成果替代传统论文具有现实需求和较高接受度，但当前制度认知不足。
2. 学生更偏好与真实企业问题紧密相关、可直接体现分析和决策能力的成果形式。
3. 高质量实践成果的关键标准集中在真实性、问题导向性、数据支撑度、规范性和可操作性。
4. 当前推进中的核心痛点不是意愿问题，而是标准不清、认可机制不明、案例和模板不足。

据此可提出以下建议：

1. 尽快形成清晰的实践成果分类标准与适用边界。
2. 公布可操作的评价指标体系和评分细则，降低学生不确定性。
3. 建设模板库、案例库与写作指导资源，提高成果产出质量。
4. 引入企业导师和实践受益方参与评价，构建多元评价机制。
5. 建立企业项目对接与数据授权支持机制，缓解真实数据获取难题。

- 在管理层面，可为学院完善实践成果管理与评价体系提供初步数据支持。
